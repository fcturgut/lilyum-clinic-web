# Photos Directory

Upload your local photos here.

Once you upload an image (for example, `my-photo.jpg`), you can reference it in your code using the path `/photos/my-photo.jpg`.

For example, in the Instagram feed:
```javascript
{
  id: 1,
  image: "/photos/my-photo.jpg",
  likes: 124,
  comments: 12
}
```
